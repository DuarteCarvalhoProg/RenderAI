Core Engine

Text prompt → seamless texture

Type any material description and RenderAI generates a 1024×1024 px seamless, tileable texture powered by Stable Diffusion via Pollinations AI — directly inside Unity, in under 1 second.

  • ✦ Prompt auto-enriched with seamless, high quality, 4k, tileable
  • ✦ 1024×1024 px output, LANCZOS resampled
  • ✦ TextureWrapMode.Repeat for seamless tiling on any surface
  • ✦ Applied asynchronously — zero frame drops
renderai — flask server

  "resolucao": 1024, "seed": 42 }

Reproducibility

Consistent results with seed control

Every generation is controlled by a seed parameter. Use the same seed and prompt to reproduce the exact same texture — essential for consistent environments across multiple Unity scenes or sessions.

  • ✦ Default seed: 42 (always reproducible)
  • ✦ Change seed to explore variations
  • ✦ Parameters: seed, steps (1–50), guidance_scale (5–12)
  • ✦ All parameters configurable in Unity Inspector

Performance

Sub-second texture generation

Tested across 9 real executions, RenderAI achieves an average of 0.842 seconds end-to-end — from Unity request to texture applied. No GPU required; all generation happens in the cloud via Pollinations AI.

  • ✦ Avg AI generation: 0.702s (83% of total)
  • ✦ Avg post-processing: 0.140s (17% of total)
  • ✦ Min total: 0.745s · Max: 0.930s
  • ✦ Consistent across texture types (stone, wood, grass)

Unity Integration

Native Unity workflow

RenderAI integrates directly into the Unity Inspector. No external tools, no window switching — just type a description, click Play, and the texture appears on your object in real time.

  • ✦ Unity 6 compatible (Universal Render Pipeline)
  • ✦ Works on any primitive or custom mesh with UVs
  • ✦ Supports Cube, Sphere, Plane and any UV-mapped object
  • ✦ Call GerarTextura() from any other C# script
