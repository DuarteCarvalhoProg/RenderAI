Pipeline

From prompt to 3D texture in 4 steps

RenderAI connects your words to Unity through a Flask server and the Pollinations AI engine — all in under a second.

01
✍

Write your prompt

Type a texture description in plain language directly in the Unity Inspector or GUI. RenderAI automatically enriches it with seamless, 4k, tileable keywords.

"brick wall" → "brick wall, seamless texture, high quality, 4k, tileable"
→
02
⚙️

Flask server processes

Unity sends an HTTP POST to a local Python/Flask server. The server builds a Pollinations AI URL with your prompt, resolution and seed parameters.

→
03
🤖

AI generates texture

Pollinations AI generates a 1024×1024 px seamless texture via Stable Diffusion in under 1 second. The Flask server resizes, converts to RGB and returns PNG.

→
04
🎮

Applied in Unity

The C# script receives the PNG, creates a Texture2D with TextureWrapMode.Repeat for seamless tiling, and applies it to the object's material — all asynchronously.
